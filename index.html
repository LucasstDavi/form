Perguntas e Resposta sobre Formulário

A) Na tag input com atributo type="text", para que serve o atributo " placeholder"?

RESPOSTA:

O atributo placeholder é uma pequena dica, uma pequena frase, uma palavra, que tem o intuito de ajudar o usuário a entender como ele deve preencher aquele formulário.

B) Na tag input com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

RESPOSTA:

especifica que este botão "checkbox" ou "radio" é selecionado por default.

C) Na tag input para que serve o atributo "value"?

RESPOSTA:

"Enviar dados", especifica-se o texto que será lido pelo leitor.

D) Que atributo deve ser utilizado para agrupar vários botões de ádio em um formulário de modo que apenas um deles fique marcado de cada vez?

RESPOSTA:

Tem que ultilizar o atributo (name) Exemplo de linha de código: (input type="radio" name="smt" id="smt").

E) Para que serve a tag label? Para que serve o atributo "for" nessa tag?

RESPOSTA:

Tag label serve para aparecer a irformação que está pedindo no formulário. O atributo "for" faz um tipo de link entre a label e o input.

F) Na tag option para que serve o atributo "selected"?

RESPOSTA:

Este atributo booleano indica que várias opções podem ser selecionadas na lista. Se não for especificado, apenas uma opção poderá ser selecionada de cada vez.

G) Nas tags de formulário para que serve o atributo "required"?

RESPOSTA:

Serve para dizer para o usúario que aquele iten do formulário é obrigatório ser respondido.

H) Na tag form para que serve o atributo "action"?

RESPOSTA:

O atributo "action" define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados.

I) Na tag form para que serve o atributo "method"?

RESPOSTA:

O atributo method define qual o método HTTP para enviar os dados (ele pode ser "GET" ou "POST").

J) Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?

RESPOSTA:

Colocar o atributo "POST".

K) Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?

RESPOSTA:

(input type=”file”) O input do tipo “file” define um campo de seleção de arquivos e um botão para upload de arquivos. Caso deseje permitir a seleção de vários artigos, pode adicionar o atributo “multiple”.
